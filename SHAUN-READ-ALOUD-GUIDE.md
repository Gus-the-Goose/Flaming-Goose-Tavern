# Flaming Goose Tavern — Read-Aloud Guide for Shaun

This guide explains how to use the new read-aloud features in The Flaming Goose Tavern app.

The old megaphone button only read a short welcome message. The new version has a better read-aloud button that can read the current page and recent table messages.

---

## 1. Open the app

Open the Flaming Goose Tavern link Anna gives you.

If you are on iPhone or iPad, use **Safari** if possible.

---

## 2. Use the read-aloud button

At the top of the app, near the other tool buttons, there is a **megaphone button**:

**📢**

Press this button to make the app read the page aloud.

It should read things like:

- the app title
- the current scene
- the latest dice result, if there is one
- recent table messages

If it starts reading and you want it to stop, use the small read-aloud control that appears near the bottom-right of the screen, or press another read-aloud button.

---

## 3. Turn on automatic reading of new messages

If you want the app to automatically read new table messages when they arrive:

1. Press the **settings button** at the top of the app.
   - It looks like this: **⚙️**
2. Find the section called **Speech**.
3. Turn on:
   - **Voiceover mode: automatically read new table messages**

After this is turned on, new table messages should be spoken aloud when they appear.

You can turn this off again any time by going back to **Settings → Speech** and unticking the same option.

---

## 4. Choose a voice

In **Settings → Speech**, there is also a **Read-aloud voice** option.

You can choose:

- Default browser voice
- Female voice if available
- Male voice if available

The exact voices depend on your phone, tablet, or computer.

---

## 5. Reading one message at a time

Each table message also has its own small **📢** button.

Press that button if you only want to hear that one message, instead of the whole page.

---

## 6. Put the app on an iPhone home screen

This is not an App Store app yet, but you can add it to your iPhone or iPad home screen so it feels more like an app.

On iPhone/iPad:

1. Open the Flaming Goose Tavern link in **Safari**.
2. Press the **Share** button.
   - It usually looks like a square with an arrow pointing up.
3. Choose **Add to Home Screen**.
4. Press **Add**.

After that, you should have a Flaming Goose Tavern icon on your home screen.

---

## 7. If speech does not work

Try these in order:

1. Make sure your device volume is turned up.
2. Make sure silent mode is not blocking audio.
3. Try pressing the **📢** button again.
4. Try closing the browser tab and opening the app again.
5. On iPhone/iPad, try Safari rather than another browser.
6. If automatic reading does not work, try the manual **📢 Read this page aloud** button.

---

## 8. What to tell Anna if something is wrong

If it still does not work, tell Anna:

- what device you are using, for example iPhone, iPad, Android phone, Windows laptop
- what browser you are using, for example Safari, Chrome, Edge
- whether the **📢** button appears
- whether pressing it does nothing, reads the wrong thing, or starts but stops too soon

That will help us fix it.
